#ExpandableLayout

* A expand/collapse layout.


|attr|format|description|
|---|:---|:---:|
|duration|Integer|Animation duration|
|clickToToggle|boolean|Click child view to dismiss|
|initialState|enum(expand/collapse)|Show the effect for first time|
